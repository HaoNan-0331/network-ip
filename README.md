# NetworkIP

网络IP/MAC监控系统 - 用于网络设备ARP表采集与IP地址管理

## 功能特性

### 设备管理
- 支持华为、H3C、Cisco、锐捷等主流网络设备
- SSH/Telnet协议连接
- 设备状态实时监控
- 设备分组管理

### ARP采集
- 手动采集：一键从所有设备采集ARP表
- 定时采集：可配置采集间隔（5分钟~24小时）
- 并发采集提升效率
- 支持多厂商命令解析

### 网段管理
- IP使用统计与可视化
- IP状态追踪（已用/弃用）
- 网段自动发现
- IP详情查询（MAC、厂商、接口等）

### 异常检测
- IP-MAC变化监控
- 变化告警与确认
- 历史记录查询
- 排除IP管理

### OUI管理
- MAC厂商数据库
- 自定义OUI条目
- 批量导入支持

### 系统设置
- 定时采集任务配置
- 数据库备份与重置
- 数据导出（CSV格式）

## 安装使用

### 方式一：直接运行（推荐）

1. 从 [Releases](https://github.com/HaoNan-0331/network-ip/releases) 下载 `NetworkIP-x.x.x-portable.zip`
2. 解压到任意目录
3. 运行 `NetworkIP.exe`

### 方式二：从源码构建

```bash
# 克隆仓库
git clone https://github.com/HaoNan-0331/network-ip.git
cd network-ip

# 安装依赖
npm install

# 开发模式
npm run dev

# 打包应用
npm run make
```

## 操作说明

### 1. 添加设备

1. 进入「设备管理」页面
2. 点击「添加设备」按钮
3. 填写设备信息：
   - 名称：设备标识名称
   - IP地址：设备管理IP
   - 厂商：选择华为/H3C/Cisco/锐捷
   - 协议：SSH或Telnet
   - 端口：默认22（SSH）或23（Telnet）
   - 用户名/密码：设备登录凭证
4. 点击「测试连接」验证
5. 保存设备

### 2. ARP采集

**手动采集：**
1. 进入「ARP采集」页面
2. 点击「全部采集」按钮
3. 等待采集完成

**定时采集：**
1. 进入「系统设置」->「定时采集」
2. 开启「启用定时采集」
3. 选择采集间隔
4. 系统将自动按间隔执行采集

### 3. 网段管理

**添加网段：**
1. 进入「网段管理」页面
2. 点击「添加网段」
3. 填写网段信息（如 192.168.1.0/24）
4. 保存后可查看IP使用情况

**查看IP详情：**
1. 选择网段
2. 点击「查看IP详情」
3. 可查看每个IP的MAC、厂商、状态、最后发现时间等

**IP状态说明：**
- **已用**：最近一次采集时发现的IP
- **弃用**：之前采集到但最近一次未发现的IP

### 4. 异常检测

当IP-MAC绑定关系发生变化时，系统会自动记录：
- 进入「异常检测」页面查看变化记录
- 可确认/忽略异常
- 支持导出记录

### 5. 数据备份

1. 进入「系统设置」->「数据库」
2. 点击「备份数据库」
3. 选择保存位置

**重置数据库：**
- 点击「重置数据库」将清除所有数据（不可恢复）

## 技术栈

- **前端**: Vue 3 + TypeScript + Element Plus
- **后端**: Electron + Node.js
- **数据库**: better-sqlite3 (SQLite)
- **网络连接**: ssh2 + telnet-client

## 系统要求

- Windows 10/11 (64位)
- 约200MB磁盘空间

## 数据存储

应用数据存储在：
```
C:\Users\<用户名>\AppData\Local\network-ip\
├── network-ip.db      # 数据库文件
└── network-ip.db-wal  # WAL日志
```

## 超时配置

  ┌────────────┬──────────┐
  │    操作    │ 超时时间 │
  ├────────────┼──────────┤
  │ SSH连接    │ 10秒     │
  ├────────────┼──────────┤
  │ Telnet连接 │ 10秒     │
  ├────────────┼──────────┤
  │ 单设备采集 │ 30秒     │
  └────────────┴──────────┘

  说明：
  - 设备连接失败时，约10秒后返回错误
  - 整个采集操作（连接+执行命令）最长30秒

## 卸载

免安装版本：
1. 删除程序目录
2. 如需清除数据，删除 `C:\Users\<用户名>\AppData\Local\network-ip\` 目录
